# Text Similarity

## Introduction

The Purpose of this Project is that given a two text we have to check how similar it is.For Similarity Checking i used Spacy ('en_core_web_sm').The Output of Similarity will be in range [0,1] where 0 is worst and 1 is best.Before Checking whether two Sentence is Similar or not You need to do  Following Steps.

- First you need to register Yourself (username,password).
- After registration you will Get 10 Tokens and one text similarity cost 1 token. you have an option to Refil your Token also.
